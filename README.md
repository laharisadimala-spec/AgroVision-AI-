# AgroVision-AI-
AI-powered crop disease detection and smart farming assistant with multilingual and voice-enabled support for sustainable agriculture.

AgroVision AI is an AI-powered smart farming platform that helps farmers detect crop diseases, receive treatment recommendations, access weather-based farming advisories, and interact with an intelligent multilingual farming assistant.

Built using:
- Next.js
- TypeScript
- Gemini AI
- Tailwind CSS
- Framer Motion
